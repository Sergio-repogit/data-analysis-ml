# Data Analysis Projects

Repositorio que recoge distintos proyectos de análisis de datos, abarcando desde preprocesamiento y modelización hasta extracción de información mediante expresiones regulares.

Cada subcarpeta contiene un proyecto independiente con su correspondiente código, memoria y conjunto de datos.

---

## Contenido del repositorio

### dataset_obesidad
Proyecto de análisis estadístico sobre datos de obesidad.

**Incluye:**
- `obesidad_cleaned` — conjunto de datos preprocesado  
- `statistical-data-analysis-report` — memoria con el análisis realizado  

**Objetivo:**  
Exploración de variables relacionadas con obesidad, análisis estadístico y modelización básica.
